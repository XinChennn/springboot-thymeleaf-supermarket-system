# 基于SpringBoot 超市管理系统

## 一、系统概要
超市管理系统总共分为两个大的模块，分别是系统模块和业务模块。其中系统模块和业务模块底下又有其子模块。

其功能包括：客户管理、供应商管理、商品管理、商品进/退货管理、商品销售(退货)管理、用户管理、角色管理等

## 二、技术选型
后端

* SpringBoot
* Shiro
* MybatisPlus

前端

* LayUI
* DTree

## 三、开发环境
Java、IDEA、Navicat、Git、Maven、MySQL
